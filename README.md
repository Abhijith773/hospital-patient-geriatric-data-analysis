# Hospital Patient Geriatric Data Analysis

## Project Overview
This project analyzes hospital patient data focusing on age groups, disease prevalence, treatment costs, outcomes, with a focus on geriatric medicine.

## Dataset
The dataset includes patient demographics, conditions, treatments, costs, and outcomes.

## Repository Structure
- `data/`: Contains raw data.
- `notebooks/`: Jupyter notebooks with analysis and visualizations.
- `scripts/`: Python scripts for data cleaning and visualizations.
- `README.md`: Project details.
- `requirements.txt`: Python dependencies.
- `.gitignore`: Ignore unnecessary files.

## How to Run
- Clone the repo
- Install dependencies: `pip install -r requirements.txt`
- Run notebook or scripts for analysis.

## Key Insights
- Majority patients are above 40 years with chronic diseases.
- High treatment cost and longer stays in elderly.
- Indicates importance of specialized geriatric medicine.
